Features

Everything you need for PDFs

15 built-in tools plus a visual editor and integrated viewer. All running natively on your machine, no internet required.

PDF Tools

15 tools, one app

Split

Split a PDF into multiple files by page ranges. Define custom output names for each part.

Merge

Combine multiple PDFs into one. Drag-and-drop to reorder before merging.

Rotate

Rotate individual pages or the entire document by 90°, 180°, or 270°.

Extract pages

Pull specific pages out of a PDF into a new file. Supports ranges and individual selections.

Reorder

Drag-and-drop interface to reorder or remove pages with live preview.

Compress

Three quality levels (extreme / recommended / low) using a 3-pass pipeline with Ghostscript and PyMuPDF.

Encrypt & Decrypt

Protect a PDF with owner and user passwords, or remove existing protection.

Watermark

Overlay a watermark or stamp PDF on selected pages, above or below content.

OCR

Recognize text in scanned PDFs using Tesseract. Supports English, Portuguese, Spanish, French, and German.

Convert

Export PDF to PNG, JPG (with custom DPI), DOCX, TXT, PPTX, XLSX, HTML, or EPUB.

Import to PDF

Convert TXT, images (PNG/JPG/BMP), Markdown, DOCX, PPTX, XLSX, HTML, or EPUB files to PDF. Supports batch import.

N-up (page layout)

Place 2, 4, 6, 9, or 16 pages per sheet. A4/A3/Letter output with portrait or landscape orientation.

Page numbers

Add page numbers in 6 positions with customizable format (e.g. "Page 1", "1/10", "1 of 10"). Choose font size and starting number.

Visual Editor

9 modes: redact, add text, insert image, highlight, sticky note, fill forms, edit existing text, signature, select. With undo/redo and continuous scroll.

Info

View metadata, page count, file size, encryption status, fonts, and creation dates.

Integrated viewer

Read PDFs the way you want

A modern reading experience built into the same app — no need for a separate viewer.

Bookmarks / TOC

Clickable side panel showing the PDF outline. Jump to any chapter or section.

Night reading mode

Invert PDF colors for comfortable reading at night. Independent from the app theme.

Full-text search

Press Ctrl+F to search across the entire PDF. Live highlights and match navigation.

Presentation (F5)

Fullscreen slideshow with arrow-key navigation, page counter, and Escape to exit.

Tabbed viewing

Open multiple PDFs in tabs and switch between them like in a browser.

Drag & drop

Drop PDFs anywhere in the window to open them. Supports multi-select.

Localized

8 languages built-in

Auto-detects your system language at startup. Switch on the fly from the toolbar.
